Memori

Memori is a simple memory game involving patterns. Follow the circles was a game developed for android, and consists of players memorizing and then repeating randomly selected patterns of circles. The game itself was developed on android studio, and uses no game engine to support itself. Thus, the code may be a little verbose. Nevertheless, the code does function and should provide for a learning experience for aspiring android developers. To access the game itself, the application Follow The Circles must be downloaded from the android store.
NOTE: The game is only available on android and can only be downloaded through the Google Play Store

Screenshots of the game

How and Why Memori Was Made

Memori was my first attempt at making a tangible application through my coding knowledge. Inspired by the power of Java, I decided to attempt to hardcode an Android application. Thus I began developing Memori, and created the game after a couple of months learning how to program with the Android APIs and Java in the context of android application. After much hard work, I finally produced a finished product that could be published, and released Memori on the Google Play Store.

Language-wise, Memori is coded with absolutely no engine or software assistance besides that of the Android APIs. Thus, Memori can be read without any knowledge other than that of Java. However, the game does require basic runtime knowledge such as locking and running threads, multithreading, and basic rendering concepts. However, the code for Memori is noticeably untidy, as it was my first attempt at a legitimate, long-term application. That being said, it should still be decently readable and didactic.
